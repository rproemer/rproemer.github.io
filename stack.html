Project Stack

This document outlines the current tools and technologies used to build, host, and maintain richardroemer.com.

Frontend

  • HTML5 / CSS3 — Custom hand-written static markup and styling
  • Responsive Design — Mobile-optimized using media queries and flexbox
  • PDF Integration — Direct link to downloadable résumé
  • Image Handling — Transparent PNG headshot for header section

Design Tools

  • Adobe Photoshop 2024 — Used to remove background and crop profile image
  • Manual Asset Optimization — Ensures small file size and transparent rendering

Email Configuration

  • Forwarding: ImprovMX — Routes emails from connect@richardroemer.com to a personal Gmail account
  • Sending: Gmail custom alias with app-specific password (SMTP auth)
  • Catch-All Enabled — Future-proofing for any address under the domain

Domain & Hosting

  • Domain Registrar: IONOS
  • DNS Configuration: A and CNAME records pointing to Render
  • Web Hosting: Render (Free Static Site Tier)
  • SSL Certificate: Automatically provisioned by Render

Development Workflow

  • Source Control: GitHub
  • Repository: rproemer.github.io
  • Editor: Visual Studio Code
  • Deploys: Manual trigger on Render after commit sync
